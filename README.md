# TMX Reader - Lector de Archivos TMX Online

Herramienta web gratuita y de código abierto para visualizar archivos TMX (Translation Memory eXchange).

## Características

- 100% gratuito y sin registro
- Procesa archivos localmente (privacidad total)
- Funciona en cualquier navegador
- Búsqueda en tiempo real
- Diseño responsivo

## Uso

1. Visita https://thegorgones.github.io/tmx-reader/
2. Selecciona tu archivo TMX
3. ¡Visualiza y busca tus traducciones!

## Uso local

También puedes descargar el código y abrirlo localmente:
